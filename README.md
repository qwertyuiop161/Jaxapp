# Jaxapp
Programming language built in CPP (with a bit of assembly for the runtime) designed to have very similar syntax to java. The whole point is to allow java programmers to continue coding with syntax they enjoy (I like the class system and java syntax a lot more). CPP has similar syntax and higher speed, but it also has much more complicated things like std::cout. This language combines the easy syntax of java with the speed of C++.
